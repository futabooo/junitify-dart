# Product Overview

junitify は、Dart テストフレームワークの JSON 出力を JUnit XML 形式に変換するコマンドラインツールです。CI/CD パイプラインやテストレポートツール（Jenkins、GitLab CI、GitHub Actions など）と Dart プロジェクトのテスト結果を統合するための橋渡しを提供します。

## Core Capabilities

1. **JSON 入力処理**: Dart テストの標準 JSON 出力をファイルまたは標準入力から読み込み
2. **JUnit XML 変換**: Dart テスト結果を JUnit XML スキーマ準拠の形式に変換
3. **柔軟な I/O**: ファイル I/O と標準入出力の両方をサポート
4. **エラーハンドリング**: 型安全なエラー処理と詳細なエラーメッセージ
5. **大規模データ対応**: 10,000+ テストケースの効率的な処理

## Target Use Cases

- **CI/CD 統合**: Dart プロジェクトのテスト結果を既存の CI/CD パイプラインに統合
- **テストレポート**: 標準化された JUnit XML 形式でテスト結果を可視化
- **マルチ言語プロジェクト**: 複数の言語で書かれたプロジェクトで統一されたテストレポート体験を提供

## Value Proposition

Dart プロジェクトのテスト結果を CI/CD エコシステムの標準的な一部として扱えるようにし、Dart 開発者が既存の DevOps ツールチェーンとシームレスに統合できるようにします。変換専用ツールとして、シンプルで信頼性の高い変換機能に焦点を当てています。
